== Description ==

Blogarate plugin allows you to have your blog post rated. The plugin shows the score you've got and the total number of viewer that have rated your post.  Full admin options available. Can styles the plugin in several ways.

= Features: =

* Full admin options for style your own Blogarate plugin. 

* Can hide avatars of raters from Blogarate. 


== Installation ==

1. Upload the "Blogarate" folder to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. Click "Setting" tab and find a link to "Blogarate" options


== Frequently Asked Questions ==
How to upload icons for the plugin?

Click "Setting" tab and find a link to "Blogarate" options, upload fields appear under "Custom icons".
After uploading the files remember to check "Use custom icons" check box. Then click on  "Save Setting" button.

How to switch off icons for the plugin?

Click "Setting" tab and find a link to "Blogarate" options, under "Custom icons" uncheck "Use custom icons" check box.
Then click on  "Save Setting" button.

What are the restriction on icons?
Icons have to be in these file format: .jpg, .gif or .png.
Icons dimensions have to be smaller than 100 x 100.
